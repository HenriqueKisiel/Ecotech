Ecotech - Sistema de gestão para uma empresa de Resíduos Eletrônicos.

Principais Funcionalidades:
- Agendamento de Coleta
- Criação de Rotas
- Pesagem do material
- Separação
- Movimentação
- Suporte a gestão (Relatórios e Dashboard)

CRUDS:
- Pessoa Física/Jurídica
- Funcionario
- Planta (Franquia)
- Estoque
- Material
- Motorista
- Caminhão
- Agendamento
- Rota
